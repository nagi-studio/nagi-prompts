# Screenshot checklist

This directory contains reviewed screenshots of the example result.

## Included

- `desktop-vr-overview.webp` — desktop VR layout, captured 2026-07-12.

Before adding a screenshot:

- capture only public-facing pages;
- remove account, admin, analytics, visitor, and unpublished information;
- export as WebP where practical;
- record the capture date and owner in `prompts/sao-blog-theme/references.md`;
- do not add private source files or design assets.

Screenshots document the result of the written Prompt. They are not required inputs for using it.
