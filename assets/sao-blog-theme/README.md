# Screenshot checklist

This directory is reserved for reviewed screenshots of the public reference result.

Before adding a screenshot:

- capture only public-facing pages;
- remove account, admin, analytics, visitor, and unpublished information;
- export as WebP where practical;
- record the capture date in `prompts/sao-blog-theme/references.md`;
- do not add private source files or design assets.

The initial repository intentionally ships without screenshots until a visual privacy and rights review has been completed.
